# WashU EEPS GitHub

This is the GitHub organization hosting code, teaching materials, and reproducible research outputs from the Department of Earth, Environmental, and Planetary Sciences (EEPS) at Washington University in St. Louis.

## What you will find here

- research software and analysis pipelines  
- course repositories and assignments  
- shared departmental tools and templates  

## Joining the organization

GitHub organizations cannot be joined automatically. Members must be invited.

To request access:

1. Create a GitHub account.
2. Open a request here:  
   https://github.com/washu-eeps/join/issues/new
3. Include your GitHub username and your role (faculty, graduate student, staff, collaborator).

An administrator will send an invitation to join the organization.
